# Rules Documentation

This directory contains documentation and examples for Claude Code rules.

## What are Rules?

Rules are instructions in `CLAUDE.md` files that guide Claude's behavior for your project.

## Structure

- `kotlin.md` - Kotlin style & best practices
- `typescript.md` - TypeScript style & best practices
- `coding-standards.md` - General code style and conventions
- `testing-guidelines.md` - Testing requirements
- `security-rules.md` - Security best practices
